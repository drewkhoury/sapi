# Examples

```

# example data
# ------------
# skills_list
# ------------
#
# hipchat_username, skill, level
# ------------------------------
# foo,  bar, 7
# foo2, bar
# foo2, baz

# add data
curl -X POST \
--header "Content-Type: application/json" \
--header "Accept: application/json" -d "{
  \"hipchat_username\": \"foo\",
  \"skill\": \"bar\"
}" "http://localhost:3000/api/skillslist?access_token=foo&access_token=foo"

# list all skills
http://localhost:3000/api/skillslist?access_token=foo

# search for skill foo
http://localhost:3000/api/skillslist?access_token=foo&filter[where][skill]=foo

# search for user foo
http://localhost:3000/api/skillslist?access_token=foo&filter[where][hipchat_username]=foo

```
